## How this site is hosted

This is a GitHub Pages site. The repo is named `vinayanharichandran.github.io` (matching the GitHub username), which tells GitHub to publish the contents at the matching URL.

To update the site, edit `index.html` and commit. Changes go live within 1–2 minutes.
